# java-tennis-match
represents a tennis match standalone application.
the start point class is TennisMatch witch expose the application api.

To facilitate the test task, the  App.java presents an automation to select between thres implementations of the project : sprint1_useCase1
, sprint1_useCase2 and sprint2_useCase2 .
I created three git tag each for a project version.

### how to execute it ?
1 mvn clean install
2 run it as simple standalone app and follow the indications.
